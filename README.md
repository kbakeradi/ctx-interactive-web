# Readme - Interactive Web

## Usage

This repo uses [Vite](https://vitejs.dev/guide/) with ReactJS-TS template.

## Installation

```bash
cd InteractiveWeb

# npm:
npm install

# yarn:
yarn install

# run as dev
npm run dev
yarn run dev
```

## Production

The below command will generate a /dist folder inside the root.

```bash
# production build
npm run build
yarn run build
```
